## Usage Instructions

### Generate Single Image
```bash
npx nano-banana-pro generate "[prompt text]"
```

### Generate All Scenes
```bash
for i in {1..10}; do
  npx nano-banana-pro generate "$(cat scene${i}.txt)"
done
```

### Recommended Settings
- **Aspect Ratio**: 16:9 (presentation slides)
- **Style**: Professional / Corporate
- **Quality**: High / Ultra
- **Format**: PNG for transparency support

### Style Guidelines
- Use ServiceNow brand colors (green: #81B5A1, dark blue: #032D42)
- Maintain corporate professional aesthetic
- Include technology visualization elements
- Ensure text-safe zones for slide overlays
